# Пятнашки (Fifteen Puzzle)

Классическая игра пятнашки для iOS и Android.

## Режимы игры
- 3×3 (8 костяшек)
- 4×4 (15 костяшек)
- 5×5 (24 костяшки)
- 6×6 (35 костяшек)

## Темы
- Дерево
- Мрамор
- Неон
- Чёрно-белая
- Пейзажи (Unsplash)
- Природа (Unsplash)
- Абстракция (Unsplash)
- Животные (Unsplash)

## Функции
- Таймер и счётчик ходов
- Топ-10 рекордов по режимам
- Звук и вибрация при победе
- Плавные анимации
- Рандомные картинки из Unsplash

## Сборка

### iOS (требуется macOS или GitHub Actions)
```bash
flutter build ios --release
```

### Android
```bash
flutter build apk --debug
```
